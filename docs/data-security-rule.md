# docs/data-security-rules.md

# 数据安全规则

本项目涉及：

- 未成年人信息
- 学校信息
- 作文原文
- 联系方式
- 投稿记录

必须优先保证数据安全。

---

# 一、敏感数据范围

以下属于敏感数据：

- 学生姓名
- 学校
- 班级
- 联系方式
- 作文正文
- 投稿记录
- 登录记录
- 微信绑定信息

---

# 二、游客限制

游客禁止查看：

- 学生真实信息
- 投稿邮箱
- 完整作文
- 联系方式
- 投稿记录

---

# 三、身份隔离

不同身份默认隔离。

禁止：

- 家长查看其他学生
- 老师查看非授权班级
- 机构跨机构访问
- 编辑查看无关投稿

---

# 四、作文数据规则

作文原文：

- 不允许用于 AI 训练
- 不允许公开展示
- 不允许被其他用户访问

AI改稿：

- 必须保存原稿
- 必须生成新版本
- 不允许覆盖原文

---

# 五、文件上传规则

上传文件必须：

- 校验用户身份
- 校验文件类型
- 限制文件大小
- 防止恶意文件

禁止：

- 任意文件上传
- 公共直链暴露敏感文件

---

# 六、账号安全规则

支持：

- 微信登录
- 手机号登录
- 账号找回
- 微信换绑

账号找回必须：

- 人工审核
- 保留审核记录

---

# 七、日志规则

敏感操作必须记录：

- 操作人
- 操作时间
- IP
- 操作内容

包括：

- 删除
- 退款
- 权限修改
- 账号冻结
- 数据导出

---

# 八、后台安全规则

后台必须：

- 登录校验
- 权限校验
- 操作日志
- 接口鉴权

禁止：

- 后台接口裸露
- 管理接口无权限
- 超级管理员共享账号

---

# 九、隐私展示规则

默认展示：

- 昵称
- 脱敏信息

禁止公开：

- 手机号
- 学校全称
- 微信信息
- 完整作文

---

# 十、导出规则

导出功能必须：

- 校验权限
- 限制范围
- 记录日志

禁止：

- 全量学生数据随意导出
