# docs/activity-rules.md

# 活动库开发规则

活动库是平台核心资产。

必须结构化。

---

# 一、活动核心字段

必须包含：

- 活动标题
- 征稿方
- 投稿邮箱
- 适合年级
- 投稿主题
- 文体
- 截止时间
- 是否有稿费
- 是否有证书
- 是否有样刊
- 投稿方式
- 原文征稿详情
- 来源链接
- 更新时间

---

# 二、展示规则

游客：

只能查看部分字段。

会员：

可查看完整内容。

---

# 三、活动详情规则

活动详情页必须包含：

- 基础信息
- 征稿详情
- 投稿方式
- AI投稿建议
- 风险提醒
- 相关推荐

---

# 四、结构化规则

字段必须结构化。

禁止：

- 大段纯文本无法筛选
- 所有内容混在正文里

---

# 五、活动状态

包括：

- 征稿中
- 即将截止
- 已截止
- 长期征稿
- 待审核
- 审核未通过

---

# 六、活动导入规则

支持：

- 后台导入
- 编辑发布
- Excel导入

必须：

- 字段校验
- 去重
- 状态校验

---

# 七、SEO/GEO规则

活动必须支持：

- SEO 页面
- GEO 页面
- AI摘要
- FAQ
- 聚合页

---

# 八、征稿方规则

征稿方必须独立。

支持：

- 征稿方主页
- 样刊展示
- 样文展示
- 历史活动
