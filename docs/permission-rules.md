# docs/permission-rules.md

# 权限规则文档

系统采用：

「账号 → 身份 → 权益」

三层权限模型。

所有权限判断都必须基于当前 identity_id，而不是单纯 user_id。

---

# 一、核心原则

1. 权限必须基于 identity_id。
2. 不同身份默认数据隔离。
3. 权益绑定身份，不绑定账号。
4. 所有敏感操作必须服务端校验。
5. 禁止只依赖前端隐藏按钮控制权限。
6. 所有敏感操作必须记录日志。
7. 自主投稿和平台代投权限必须分开判断。
8. 用户前台权限和运营后台权限必须分开。

# 三、游客权限

游客可以：

- 查看首页
- 查看SEO/GEO页面
- 查看部分活动列表
- 查看活动摘要
- 查看投稿攻略

游客禁止：

- 查看完整活动详情
- 查看投稿邮箱
- 查看详细投稿方式
- 使用AI改稿
- 使用AI推荐
- 上传作文
- 创建学生
- 创建自主投稿记录
- 申请平台代投
- 查看投稿记录
- 查看成长档案

---

# 四、家长权限

家长可以：

- 管理自己的孩子
- 上传自己孩子的作文
- 使用AI改稿
- 使用AI推荐活动
- 查看已购买年级范围内的完整活动
- 查看投稿邮箱
- 查看投稿方式
- 创建自主投稿记录
- 申请平台代投
- 查看自己的自主投稿记录
- 查看自己的平台代投记录
- 查看自己的投稿截图
- 查看自己孩子的成长档案

家长禁止：

- 查看其他家长的孩子
- 查看其他用户作文
- 查看其他用户投稿记录
- 查看后台运营备注
- 修改平台代投后台状态
- 上传平台代投截图

---

# 五、个人老师权限

老师可以：

- 创建和管理自己的学生
- 上传学生作文
- 批量上传作文
- 使用AI改稿
- 使用AI推荐活动
- 查看投稿方式
- 创建自主投稿记录
- 批量创建自主投稿记录
- 申请平台代投
- 批量申请平台代投
- 查看自己学生的投稿进度
- 查看自己学生的成长档案

老师禁止：

- 查看非自己管理的学生
- 查看其他老师的投稿记录
- 查看后台运营备注
- 修改平台代投后台状态

---

# 六、机构管理员权限

机构管理员可以：

- 管理机构信息
- 管理机构老师
- 管理机构学生
- 查看机构订单
- 批量导入学生
- 批量上传作文
- 批量使用AI推荐
- 批量查看投稿方式
- 批量创建自主投稿记录
- 批量申请平台代投
- 查看机构内投稿数据
- 查看机构内成长档案

机构管理员禁止：

- 查看其他机构数据
- 越权管理其他机构老师
- 查看平台运营内部备注
- 修改平台代投后台状态

---

# 七、机构老师权限

机构老师可以：

- 查看授权班级
- 管理授权学生
- 上传授权学生作文
- 使用AI推荐
- 创建自主投稿记录
- 申请平台代投
- 查看授权学生投稿进度

机构老师禁止：

- 查看非授权班级
- 查看非授权学生
- 查看机构财务敏感信息，除非机构管理员授权
- 查看平台运营内部备注

---

# 八、编辑权限

编辑可以：

- 发布征稿活动
- 管理自己发布的活动
- 查看自己活动的来稿
- 更新自己活动的录用状态
- 查看自己活动相关投稿情况

编辑禁止：

- 查看其他编辑活动后台数据
- 查看平台代投运营备注
- 查看用户完整隐私信息
- 查看非自己活动的投稿记录
- 管理会员、订单、退款

---

# 九、平台运营权限

平台运营可以：

- 查看被授权的后台模块
- 审核活动
- 管理活动库
- 查看平台代投任务池
- 接单或被分配代投任务
- 更新代投后台状态
- 上传投稿截图
- 填写运营备注
- 处理投稿失败
- 进行OCR识别复核
- 处理部分售后

平台运营禁止：

- 修改超级管理员权限
- 查看未授权的敏感财务信息
- 删除核心数据，除非被单独授权
- 导出全量学生数据，除非被单独授权

# 十、超级管理员权限

包括：

- 用户管理

- 权益管理
- 会员管理
- 活动管理
- 订单管理
- 代投任务管理
- 运营管理
- 系统设置
- 数据导出
- 权限配置

---

# 十一、自主投稿权限

自主投稿属于用户自助流程。

可以创建自主投稿记录的身份：

- 家长
- 老师
- 机构老师
- 机构管理员

前提：

- 已登录
- 当前身份有对应学生/作文权限
- 当前身份具备查看活动完整详情或投稿方式的权益

用户可以维护：

- 是否已投稿
- 投稿时间
- 用户备注
- 投稿结果

用户不可修改：

- 活动原始信息
- 投稿邮箱原始信息
- 其他用户投稿记录

---

# 十二、平台代投权限

平台代投属于增值服务与运营工作流。

用户侧可以：

- 选择作文
- 选择活动
- 申请代投
- 支付
- 查看前台状态
- 查看用户可见截图
- 查看用户可见备注

用户侧禁止：

- 修改后台运营状态
- 上传运营截图
- 查看内部备注
- 查看平台邮箱账号配置

运营侧可以：

- 查看代投任务池
- 接单
- 分单
- 更新后台状态
- 上传投稿截图
- 填写运营备注
- 记录失败原因
- 关闭任务

管理员可以：

- 查看全部代投任务
- 调整任务归属
- 处理异常任务
- 查看完整日志

---

# 十三、资源归属规则

家长：

只能查看自己孩子相关数据。

个人老师：

只能查看自己创建或被授权的学生。

机构老师：

只能查看机构授权范围内的班级与学生。

机构管理员：

只能查看本机构数据。

编辑：

只能查看自己活动相关数据。

运营：

只能查看被授权模块或被分配任务。

管理员：

可以查看全量数据。

# 十五、危险操作规则

以下操作必须二次确认：

- 删除作文
- 删除学生
- 删除活动
- 删除投稿记录
- 删除代投任务
- 批量删除
- 退款
- 修改会员权益
- 冻结账号
- 修改运营人员权限
- 导出敏感数据

---

# 十六、后台接口规则

所有后台接口必须：

- 校验登录状态
- 校验当前 identity_id
- 校验角色权限
- 校验资源归属
- 记录操作日志

禁止：

- 只在前端判断权限
- 只隐藏按钮不做接口校验
- 通过 user_id 直接跨身份访问数据
