# 视觉系统规范

整体视觉方向：

- 专业
- 清爽
- 有教育服务感
- 有平台可信感
- 有 AI 工具感
- 有 SaaS 管理感

禁止：

- 幼稚卡通风
- 资讯站风格
- 花哨特效

---

# 一、前台视觉方向

前台核心目标：

帮助用户快速完成：

查活动 → AI分析 → 获取投稿方式 → 投稿 → 跟踪结果

首页必须体现：

- 活动数据库
- AI投稿助手
- 自主投稿
- 平台代投
- 成长档案

---

# 二、后台视觉方向

后台属于：

SaaS 管理工作台。

重点：

- 表格
- 筛选
- 状态流转
- 任务处理
- 运营工作流
