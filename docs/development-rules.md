# docs/development-rules.md

# 开发规则文档

---

# 一、核心开发原则

1. 先跑通MVP。
2. 先跑通自主投稿。
3. 再做平台代投。
4. 先统一数据结构。
5. 再开发复杂功能。
6. 所有状态必须可追踪。
7. 所有权限必须统一管理。
8. 所有敏感数据必须校验资源归属。
9. AI功能不得覆盖原文。
10. 用户前台和运营后台逻辑必须分离。

---

# 三、页面开发规则

开发任何页面前，必须先输出：

- 页面名称
- 页面路由
- 可见角色
- 页面目标
- 核心模块
- 涉及数据表
- 涉及权限
- 涉及状态
- 空状态
- 加载状态
- 错误状态
- 移动端适配
- 是否属于MVP

禁止直接开始写代码。

---

# 四、组件开发规则

必须优先沉淀通用组件。

基础组件：

- Button
- Card
- Table
- Form
- Dialog
- Badge
- Tabs
- Drawer
- Pagination

业务组件：

- ActivityCard

- AiResultPanel
- SelfSubmissionCard
- AgentSubmissionTaskCard
- SubmissionStatusTimeline
- ScreenshotGallery
- IdentitySwitcher

- PaywallBlock

禁止：

- 每个页面重复写类似组件
- 状态标签样式不统一
- 表格筛选逻辑重复写

---

# 五、权限开发规则

禁止：

- 在 JSX 中散落权限判断
- 只做前端隐藏
- 不校验资源归属
- 用 user_id 替代 identity_id 判断权益

必须：

- 服务端校验
- identity_id 校验
- 资源归属校验
- 使用统一权限函数

推荐权限函数：

- canViewFullActivity()
- canViewSubmissionEmail()
- canUseAiRewrite()
- canUseAiRecommend()
- canCreateStudent()
- canCreateSelfSubmission()
- canApplyAgentSubmission()
- canViewAgentSubmissionTask()
- canHandleAgentSubmissionTask()
- canUploadSubmissionScreenshot()
- canAccessAdmin()
- canAccessEditor()

---

# 六、AI开发规则

AI必须：

- 保存调用记录
- 保存模型名称
- 保存输出摘要
- 保存作文版本
- 标记AI生成

禁止：

- 覆盖原稿
- 自动替换最终稿
- 自动投稿
- 使用作文训练模型
- 不提示AI结果仅供参考

---

# 七、投稿系统开发规则

系统支持两条路线：

1. 自主投稿
2. 平台人工代投

两套流程必须分离。

自主投稿重点：

- 用户查看投稿方式
- 用户自行投稿
- 用户记录投稿

平台代投重点：

- 用户付费申请
- 运营人工投稿
- 运营上传截图
- 用户查看进度

禁止：

- 把自主投稿和平台代投混成一个状态
- 把后台运营状态全部暴露给用户
- 代投无截图、无日志、无状态记录

---

# 八、运营工作流规则

平台代投属于运营协同工作流。

必须支持：

- 任务池
- 分单
- 接单
- 投稿截图
- 状态日志
- 失败原因
- 运营备注
- 用户可见备注
- 超时提醒

状态变化必须记录：

- 操作人
- 操作时间
- 原状态
- 新状态
- 备注

---

# 九、数据安全规则

必须：

- 学生数据隔离
- 作文数据隔离
- 文件权限校验
- 操作日志
- 敏感信息脱敏
- 导出权限控制

禁止：

- 游客查看敏感信息
- 跨身份访问学生数据
- 公共直链暴露投稿截图
- 后台接口无鉴权

---

# 十、活动库开发规则

活动字段必须结构化。

必须支持：

- 年级筛选
- 主题筛选
- 是否有稿费
- 是否有证书
- 是否有样刊
- 是否支持自主投稿
- 是否支持平台代投

游客可见：

- 活动标题
- 活动摘要
- 适合年级
- 是否有稿费/证书/样刊

会员可见：

- 投稿邮箱
- 投稿方式
- 完整征稿详情
- 详细AI建议

---

# 十一、数据库开发规则

修改数据库前必须说明：

- 新增表
- 新增字段
- 修改原因
- 影响模块
- 是否需要迁移数据

禁止：

- 未说明直接改 schema
- 删除字段不说明影响
- 把临时 mock 字段当正式字段

---

# 十二、禁止事项

禁止：

- 一次性开发所有模块
- 无权限系统
- 无日志系统
- 无状态流转
- 写死数据结构
- 页面只做静态展示
- AI覆盖原文
- 代投无截图
- 代投无运营日志
- 忽略移动端基本适配
